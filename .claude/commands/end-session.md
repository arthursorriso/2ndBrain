Close the session by consolidating everything that happened in this conversation.

1. Review the conversation: what was done, discussed, decided, and learned.
2. Update affected notes: projects (log + current state + next steps), ideas, people, tutorials. Add dated entries; never overwrite history.
3. Record new decisions in `90-memory/decisions.md` and learnings in `90-memory/learnings.md` (format: `- YYYY-MM-DD — text — context/link`).
4. Update `90-memory/current-state.md`: current focus, pending items (new and resolved), next steps.
5. Append what was done to today's daily (`60-journal/dailies/`); create it if it doesn't exist.
6. Finish with a 3-5 line summary: what was updated and where we left off, so the next session starts from the right place.
