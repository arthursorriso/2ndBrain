Generate the daily briefing. Do not create or edit files; read and present only.

1. Read `90-memory/current-state.md`.
2. Read `60-journal/weeks/<current week>.md` (`date +%G-W%V`) — "Week goals" section.
3. Read the most recent daily in `60-journal/dailies/` ("Tomorrow" and "Blockers" sections).
4. Collect open action items: unchecked boxes in meetings from the last 14 days and in "Next steps" of projects with `status: active`.
5. Check whether `00-inbox/` has unprocessed items.
6. Present: **Week goals** (progress and what's left; if none defined, suggest /goals) · **Today's focus** (from current-state + last daily's tomorrow) · **Open items** (per project, with source) · **Blockers** · **Inbox** (if items exist, suggest /process-inbox) · **Commitments to people**.
7. End with one concrete suggestion of where to start the day, prioritizing whatever unblocks goals that are behind.
