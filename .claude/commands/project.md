description: Create a new project or update an existing one
---

Project: "$ARGUMENTS"

1. Look in `10-projects/` (and `20-ideas/`) for a matching note.
2. **If it exists:** ask for/receive the update, add a dated entry to the "Update log", refresh "Current state" and "Next steps". Never erase history.
3. **If it doesn't:** create it from `templates/project.md`, asking only the essentials (goal, state, next steps, people).
4. **If it comes from an idea:** set the idea to `status: became-project` and link both notes.
5. Link related people and meetings with wikilinks.
6. Update `90-memory/current-state.md` (active projects / focus).
