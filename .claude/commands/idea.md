Capture the idea: "$ARGUMENTS"

1. Check whether a similar idea already exists in `20-ideas/` — if so, offer to enrich it instead of duplicating.
2. Create `20-ideas/<slug>.md` from `templates/idea.md`, filling in what you can from the input.
3. Connect it to related projects/ideas/meetings via wikilinks.
4. Don't interrogate the user: capture what was given, leave sections empty for later. One question at most, if essential.
